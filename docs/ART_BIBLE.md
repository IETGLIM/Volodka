### 2.2 Динамическое освещение

**Приоритет: ВЫСОКИЙ**

```typescript
// Техническая реализация (React Three Fiber)
// Требуется добавить:

1. Volumetric lighting (объёмный свет)
   - God rays через окна
   - Свет от уличных фонарей
   - Отблески экранов мониторов

2. Dynamic light sources
   - Мерцание неоновых вывесок
   - Пульсация в такт музыке (опционально)
   - Источники света, реагирующие на действия игрока

3. Screen-space reflections
   - Отражения на мокром асфальте
   - Отражения в витринах
```

### 2.3 Post-processing эффекты

**Обязательно:**
- [ ] Bloom (свечение ярких областей)
- [ ] Chromatic aberration (хроматические аберрации при низком stability)
- [ ] Vignette (виньетирование для интимности)
- [ ] Film grain (зернистость плёнки)
- [ ] Color grading (LUT для разных сцен)

**Реализация через @react-three/postprocessing:**


### 3.1 Частицы

**Приоритет: ВЫСОКИЙ**

| Эффект | Описание | Частота |
|--------|----------|---------|
| Снег | Крупные хлопья, медленное падение | Зимние сцены |
| Дождь | Мелкие капли, отражения | Переходные сцены |
| Пыль | Моточки в лучах света | Старые интерьеры |
| Искры | Электрические разряды | Киберпанк-элементы |
| Пепел | Обгоревшие страницы | Сцены разрушения |
| Звёздная пыль | Космические частицы | Сны, видения |

### 3.2 Shader-эффекты

**Для React Three Fiber:**

```glsl
// Glitch shader для низкого stability
// Расположение: src/shaders/glitch.glsl

uniform float time;
uniform float intensity;
varying vec2 vUv;

void main() {
  vec2 uv = vUv;
  uv.x += sin(uv.y * 100.0 + time) * intensity * 0.01;
  // ... continue shader
}
```

### 3.2 Эффект старой плёнки

**Приоритет: СРЕДНИЙ**

- Царапины и пыль на плёнке
- Случайные пропуски кадров
- Мягкое размытие по краям
- Сепия/чб для воспоминаний

### 3.3 Голограммы и интерфейсы

**Приоритет: ВЫСОКИЙ (киберпанк-элемент)**

## 4. КОМПОЗИЦИЯ КАДРА

### 4.1 Типы планов

| План | Использование | Пример |
|------|---------------|--------|
| Крупный | Эмоции, интроспекция | Глаза героя, руки с ручкой |
| Средний | Диалоги | Герой за столом |
| Общий | Локации | Кухня, город |
| Деталь | Символы | Текст на бумаге, фото |

### 4.2 Правила кадрирования

1. **Правило третей** — основной принцип
2. **Симметрия** — для сцен сновидений
3. **Нидерландский угол** — для нестабильности (низкий stability)
4. **Глубина кадра** — слои: передний план, основной, фон

### 4.3 Камера в 3D

**Текущая позиция:** `[0, 3, 8]`, FOV: `50`

**Рекомендуемые улучшения:**
- [ ] Cinematic camera movements (dolly, pan)
- [ ] Depth of field (размытие фона)
- [ ] Camera shake при эмоциональных сценах
- [ ] Smooth transitions между сценами

### 5.1 Список необходимых текстур

- [ ] Дерево (старый стол, паркет) — PBR: diffuse, normal, roughness

**Экстерьеры:**
- [ ] Асфальт мокрый/сухой

## 10. ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ

### 10.1 Оптимизация для Vercel

- Размер bundle < 500KB (gzipped)
- Изображения: WebP с fallback на JPEG
- 3D модели: GLTF с Draco compression
- Lazy loading для больших ассетов

### 10.2 Производительность

- 60 FPS на среднем железе
- Время загрузки < 3 секунды
- Responsive: mobile-first дизайн
- Доступность: WCAG 2.1 AA
