# `public/textures/`

Сюда кладите **PNG / WebP / KTX2** для `THREE.TextureLoader` (обои, декалы, ковры). Пока интерьеры Володьки и Заремы используют **процедурные `CanvasTexture`** в коде — внешние файлы не обязательны.

Соглашение:

- Имена в **snake_case**, без пробелов: `volodka_wall_grid.webp`.
- Размер степени двойки **512–2048** для мипмапов; для UI-плашек допустимо **640×200** и т.п.
- После добавления текстуры подключайте через **`useLoader(TextureLoader, '/textures/...')`** или статический импорт из `public/`.
